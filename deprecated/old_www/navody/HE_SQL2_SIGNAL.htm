Pøíkaz SIGNAL (SQL)

pøíkaz_SIGNAL ::= SIGNAL identifikátor_výjimky;

Pøíkaz SIGNAL vyvolá zadanou uživatelem definovanou výjimku. Výjimka musí být pøedem deklarována. Je-li deklarován handler pro tuto výjimku, handler se provede a v provádìní pøíkazù se pokraèuje v závislosti druhu handleru a místì, kde byl nalezen.

Pokud se pro zadanou výjimku nenajde handler na žádné úrovni, nastane chybový sqlstate 45000 (SQ_UNHANDLED_USER_EXCEPT).


Pøíklad použití: 

PROCEDURE `POKUS_SIGNAL`();
BEGIN
  DECLARE vyskok CONDITION;
  DECLARE CONTINUE HANDLER FOR vyskok BEGIN 
    CALL log_write("NULL - uvnitø continue handleru");
    ...
  END;

  FOR radek AS c INSENSITIVE CURSOR FOR
   SELECT * FROM Firmy
  DO BEGIN
    IF radek.adresa IS NULL THEN SIGNAL vyskok; 
    ELSE 
      ...
      CALL log_write(radek.adresa);
    END IF;
  END;
  END FOR;

END
