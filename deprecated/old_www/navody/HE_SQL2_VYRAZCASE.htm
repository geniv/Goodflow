Podmínìný výraz CASE (SQL)

výraz_CASE ::= jednoduchý_CASE | podmínkový_CASE
jednoduchý_CASE ::= CASE výraz1 { WHEN výraz2 THEN výraz3 }… [ ELSE výraz4 ] END
podmínkový_CASE ::= CASE { WHEN podmínka THEN výraz3 }… [ ELSE výraz4 ] END

Výraz CASE slouží k výbìru hodnoty jednoho z výrazù na základì udaných podmínek. V jednoduchém CASE se zleva doprava hledá vìtev WHEN, v níž výraz2 má stejnou hodnotu jako výraz1. V podmínkovém CASE se zleva doprava hledá vìtev WHEN, v níž je podmínka splnìná. Je-li nalezena vyhovující vìtev, pak hodnotou výrazu CASE se stane hodnota výrazu3 v této vìtvi. Pokud žádná vìtev nevyhoví, je hodnotou výrazu CASE hodnota výrazu4, a není-li výraz4 uveden, pak NULL.
