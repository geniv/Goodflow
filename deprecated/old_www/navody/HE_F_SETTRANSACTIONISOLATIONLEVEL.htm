Set_transaction_isolation_level



BOOL [cd_]Set_transaction_isolation_level([cdp_t cdp,] t_isolation level);
function [cd_]Set_transaction_isolation_level([cdp : cdp_t;] level : integer) : Boolean;

Parametry

[cdp kontextová promìnná klienta]
level úroveò izolace transakce

Od verze:

6.0

Popis

Funkce nastaví úroveò izolace probíhající transakce a všech následujících implicitních transakcí. Bez zavolání této funkce je nastavena hodnota READ_COMMITTED (= 1). Více o izolaci transakcí se dozvíte ve stránkách nápovìdy k jazyku SQL. Funkce odpovídá SQL pøíkazu SET TRANSACTION.

Parametr level mùže nabýt následujících hodnot:

READ_UNCOMMITTED N/A
READ_COMMITTED 1
REPEATABLE_READ N/A
SERIALIZABLE 3

Takto nastavené vlastností transakce nejen pro nejbližší transakci, nýbrž až do pøíštího nastavení vlastností transakce touto funkcí.

Použití transakcí z klientského jazyka je velmi omezené.

Hodnota funkce

Funkce vrací TRUE pøi chybì
