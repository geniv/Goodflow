Predikát IS NULL (SQL)

predikát_IS_NULL ::= výraz IS [ NOT ] NULL

Predikát IS NULL slouží k porovnávání s prázdnou hodnotou NULL, predikát IS NOT NULL k porovnání s neprázdnou hodnotou. Dle normy SQL není porovnání ve tvaru ”výraz = NULL” pøípustné (i když 602SQL je pøipouští).

Pro øetìzce je hodnota NULL totožná s prázdným øetìzcem.


Pøíklad použití:

Vybrat nezaplacené faktury:

SELECT *
FROM FAKTURY
WHERE zaplaceno IS NULL
