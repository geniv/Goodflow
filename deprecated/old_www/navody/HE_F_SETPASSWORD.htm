BOOL [cd_]Set_password([cdp_t cdp], const char * username, const char * password);
function [cd_]Set_password([cdp : cdp_t]; username, password : PChar) : Boolean;
FUNCTION Set_password(IN username CHAR(31), IN password CHAR(100)) RETURNS BIT;

Parametry

username jméno uživatele, jehož heslo se mìní
password heslo
[cdp kontextová promìnná klienta]

Popis

Funkce nastaví nové heslo zadanému uživateli. Heslo smí mít nejvýše 100 znakù.

Heslo anonymního uživatele nelze zmìnit.

Funkci mùže použít každý uživatel sám pro sebe. Heslo jiného uživatele mùže zmìnit pouze bezpeènostní správce.

Hodnota funkce

Funkce vrací pøi úspìchu FALSE, pøi neúspìchu TRUE.
