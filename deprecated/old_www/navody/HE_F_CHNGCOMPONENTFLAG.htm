Chng_component_flag



BOOL Chng_component_flag(cdp_t cdp, tcateg cat, const char * name, int mask, int setbit);
function Chng_component_flag(cdp : cdp_t; cat : tcateg; var name : PChar; mask, setbit : integer) : Boolean;

Parametry

cat kategorie objektu pro nìjž se pøíznaky nastavují
name jméno objektu pro nìjž se pøíznaky nastavují
mask bity, které se mají nastavovat
setbit hodnota nastavovaných bitù
[cdp kontextová promìnná klienta]

Od verze:

5.0

Popis

Funkce Chng_component_flag nastaví pøíznaky pro objekt kategorie cat jménem name. Nastaví ty bity, které mají hodnotu 1 v parametru mask na tu hodnotu, kterou mají v parametru setbit. Lze nastavovat, zda se objekt má exportovat pøi exportu aplikace (bit 64), zda se mají exportovat data z tabulky (bit 16) a zda objekt je startovním objektem aplikace (bit 16).

Hodnota funkce

Funkce vrací pøi chybì TRUE.
