Funkce NULLIF (SQL)

NULLIF ( výraz1, výraz2 )

Funkce NULLIF vrací hodnotu NULL pokud se hodnoty obou výrazù rovnají, jinak vrací hodnotu prvního výrazu. Výraz1 a výraz2 musí být takových typù, aby je bylo možno porovnat.

Jedná se v podstatì o zjednodušení zápisu využívajícího podmínìný výraz CASE:

CASE 
  WHEN výraz1=výraz2 THEN NULL
  ELSE výraz1
END

Poznámka: Jeden z dùvodù zavedení je historický - pokud starší databáze nemìly zavedenu hodnotu NULL, musely tuto hodnotu reprezentovat jiným zpùsobem (napø. -1 pro plat apod.). Pøi pøechodu na SQL, které vyžaduje hodnotu NULL, se tyto hodnoty musí transformovat, napø. pomocí funkce NULLIF:

... NULLIF(plat, -1) ...
